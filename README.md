Инструкция по запуску:

1) Скопировать .env.local в .env
2) Получить ключ для пользованием api KINOPOISK.
3) Скопировать ключ в переменную KINOPOISK_API_TOKEN в файле .env
4) Параметр START_SCRIPT используется для отладки и имеет два значений (without_debug, with_debug)
5) Параметр DEBUG_IP ip удалённого отладчика к которому подключается приложение 
6) Параметр DEBUG_PORT port удалённого отладчика к которому подключается приложение
7) Для запуска приложения используется скрипт external_start
8) Заходить сюда http://localhost:8181/

9) Обновление базы происходит каждую полночь: @Scheduled(cron = "0 0 0 * * *")
